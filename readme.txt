Allow guests of your blog create posts without signup

== Description ==

With that plugin you can put a form on your blog and allow any guest create new posts without signup or login. You can also edit the plugin options to modify user, post status and category to new posts. Another interesting feature included in the plugin is the option to limit the maximum number of posts creation from same IP.

== Installation ==

Follow the below steps to intall the plugin:
1- Send the folder publish-anonymous-posts to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin Publish Anonymous Posts
3- Now, in the Settings box, a new link will be shown "Publish Anonymous Posts" Click on it to go to pluin options page.
4- Change the options you want and click on button Update
5- Now, to show the form on any post, just put the following code: 
[showform]
6- Alternatively you can do a directy call to the function display_pap() using PHP. See the example below:
echo display_pap();
